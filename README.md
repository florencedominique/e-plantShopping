Paradise Nursery is a simple e-commerce React application for buying houseplants.
It uses React, React Router, and Redux Toolkit to manage state and navigation.

## Features
- Browse plants by category
- Add plants to cart
- Increase, decrease, and remove items from cart
- Dynamic cart item counter
- Simple checkout placeholder

## Technologies Used
- React
- Redux Toolkit
- React Router DOM
- CSS
